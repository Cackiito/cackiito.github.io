Exemplo de configurações para texto com CSS

Códigos para definição de fonte

font-family------ define a fonte a ser exibida no navegador

font-style ----- define se a fonte sera normal, italic ou oblique(força a fonte ficar em italico mesmo que fique ilegivel)

font-variant ----- Small Caps(faz com que as letras tenham um mesmo estilo das letras maiusculas só que menores) normal (é a letra normal).

font-weight -------- define a quantiade de negrito de uma fonte, seus atributos são: normal, bold, bolder, lighter, ou valores de 100-900

font-size:------ configura o tamanho da font, pode se confuigurar atravez de "px" "%" "pt" "em".


Alinhamento de Texto

p{text-indent: 100px;--------------- espaçamento inicial do paragrafo.
text-align: center;----------------centraliza o centro, justifica, a esquerda ou a direita.}


Decorações de texto

text-decoration: oveline;
line-through;
undeline;
underline overline;


Espaçamento entre Letras

letter-spacing: varivelpx; ---------------- configura o espaçamento entre letras.


tranformações de texto

text-transform: capitalize; ----- deixa a primeira letra da palavra maiuscula;
uppercase; ------ deixa tudo em letra maiuscula;
lowercase; ------ deixa tudo em letra minuscula;
none; --------- deixa do jeito que está;
