Exemplo de configurações para a imagem com CSS

css é folha de estilo de cascata

O Cascading Style Sheets (CSS) é uma "folha de estilo" composta por “camadas” e utilizada para definir a apresentação (aparência) em páginas da internet que adotam para o seu desenvolvimento linguagens de marcação (como XML, HTML e XHTML).


codigos para repetição de imagem de fundo:


background-repeat: repeat-x;
background-repeat: repeat-y;
background-repeat: repeat;
background-repeat: no-repeat;


codigos para o fundo da pagina

background-attachment: fixed;---------faz o fundo ficar fixado.
background-attachment: scroll;---------segue o movimento do scroll.

Codigos para a posição de imagem de fundo:

background-position: 2cm 2cm
background-position: 50% 25%
background-position: top right

Codigos para definição de fonte

font-family------ define a fonte a ser exibida no navegador


font-style ----- define se a fonte sera normal, italic ou oblique(força a fonte ficar em italico mesmo que fique ilegivel)


font-variant ----- Small Caps(faz com que as letras tenham um mesmo estilo das letras maiusculas só que menores)
normal (é a letra normal).


font-weight -------- define a quantiade de negrito de uma fonte, seus atributos são: normal, bold, bolder, lighter, ou valores de 100-900

font-size:------ configura o tamanho da font, pode se confuigurar atravez de "px" "%" "pt" "em".
